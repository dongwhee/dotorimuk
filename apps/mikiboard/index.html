Mikiboard

적은 키로 한글을 조합하는 단모음 계열 커스텀 키보드

Mikiboard는 표준 두벌식 대신 단모음 계열 배열을 사용하는 서드파티 한글 키보드입니다. 더 적은 키로 초·중·종성을 조합해, 작은 화면에서도 편안하게 입력할 수 있도록 설계했습니다.

주요 기능

  • 한글 조합 엔진 — 겹받침, 복합 모음(ㅘ·ㅙ·ㅢ 등), 쌍자음을 실시간으로 정확하게 조합합니다.
  • 선택 가능한 자판 — 단모음·뉴단모음 한글 배열과 QWERTY·8×3 영문 배열 중에서 고를 수 있습니다.
  • 입력 감도 조절 — 이중모음·쌍자음을 인식하는 시간(0.15~0.6초)을 취향에 맞게 설정합니다.
  • 햅틱 피드백 — 키 입력 시 진동 강도를 조절할 수 있습니다.
  • 편의 기능 — 스페이스 두 번으로 마침표 입력 등 작은 단축을 제공합니다.
